1 Introduction

This document describes the observations while feedback portal v2.1.0 is accessed with different bandwidth data.

2 Vocabulary

  • Throattling
    • In google-chrome browser throattling is an option which will allows to set different levels of bandwidth for loading the web page
    • Throattling helps to check web pages performance with variant of network speed.
